# 📋 Консольный планировщик задач

Простой консольный менеджер задач с тегами на Python.  
Хранит данные в JSON-файле, поддерживает добавление, отметку выполнения, удаление и очистку выполненных задач.

---

## ✨ Возможности

- ✅ Добавление задач с тегами
- 📋 Просмотр списка всех задач
- ✅ Отметка задачи как выполненной
- 🗑️ Удаление задачи
- 🧹 Очистка всех выполненных задач
- 💾 Автоматическое сохранение в `tasks.json`
- 🛡️ Обработка ошибок (пустой ввод, неверный ID и т.д.)
